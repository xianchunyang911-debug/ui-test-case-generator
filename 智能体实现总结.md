## 🚀 如何搭建智能体

### 方式1: Kiro Hook（推荐）⭐

**优势**：
- 自动触发，无需手动操作
- 集成到工作流
- 实时生成

**实现步骤**：
1. 打开Kiro命令面板（Cmd+Shift+P）
2. 搜索 "Open Kiro Hook UI"
3. 创建新Hook，配置如下：
   - Name: `UI走查用例生成`
   - Trigger: `File Save`
   - Pattern: `*需求文档*.md`
   - Prompt: 见 `智能体快速实现指南.md`
4. 保存Hook
5. 测试：保存需求文档，观察是否自动生成

**详细步骤**：参考 `智能体快速实现指南.md` 步骤2

### 方式2: Chat命令

**优势**：
- 灵活控制
- 可以指定参数
- 适合一次性使用

**使用方法**：
在Kiro Chat中输入：
```
#ui-test-gen 请根据 #需求文档.md 生成UI走查用例
```

或者：
```
请根据需求文档.md生成UI走查用例，使用Excel多Sheet格式
```

### 方式3: 自定义命令

**优势**：
- 快捷键触发
- 集成到编辑器
- 适合频繁使用

**实现步骤**：
1. 打开Kiro设置
2. 添加自定义命令
3. 绑定快捷键
4. 使用命令面板或快捷键触发

**详细步骤**：参考 `智能体快速实现指南.md` 方式3

### 方式4: Python脚本

**优势**：
- 批量处理
- 命令行集成
- CI/CD集成

**使用方法**：
```bash
python3 csv_to_excel_multi_sheet.py
```
