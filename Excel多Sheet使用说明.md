## 使用方法

### 1. 打开文件
使用Microsoft Excel、WPS表格或其他支持.xlsx格式的软件打开文件。

### 2. 查看用例汇总
- 点击"用例汇总"Sheet
- 查看各模块的用例数量和优先级分布
- 了解整体走查范围

### 3. 执行走查
按照以下步骤执行UI走查：

#### 方式一：按模块走查
1. 从第2个Sheet开始，逐个模块执行
2. 按照用例编号顺序，逐条检查UI元素
3. 在"是否通过"列选择测试结果：
   - **是**：UI符合预期，通过验证
   - **否**：UI不符合预期，存在问题
   - **待测试**：尚未测试（默认值）
4. 在"截图/备注"列记录问题描述或截图路径

> 💡 **提示**："是否通过"列有下拉选择，点击单元格右侧的下拉箭头即可选择

#### 方式二：按优先级走查
1. 使用Excel的筛选功能，筛选"高优先级"用例
2. 优先执行高优先级用例
3. 再执行中、低优先级用例

### 4. 记录问题
发现问题时，在"截图/备注"列记录：
- 问题描述
- 截图文件名（如：UI-BUG-001.png）
- 严重程度（高/中/低）
- 建议修复方案

### 5. 查看进度
"用例汇总"Sheet会**自动更新**走查进度：
- **完成数量**：自动统计各模块中"是否通过"列选择了"是"或"否"的用例数
- **完成率**：自动计算完成数量/用例数量的百分比

> 💡 **自动更新**：每次在各模块Sheet中选择"是"或"否"后，返回"用例汇总"Sheet即可看到最新进度

## Excel特性使用

### 冻结窗格
- 每个Sheet都已冻结首行（表头）和首列（用例编号）
- 滚动时表头和用例编号始终可见

### 优先级颜色标识
- 🔴 高优先级：红色背景
- 🟡 中优先级：黄色背景
- 🟢 低优先级：绿色背景

### 下拉选择
- "是否通过"列有下拉选择：待测试、是、否
- 点击单元格右侧的下拉箭头即可选择
- 选择后会自动更新"用例汇总"的完成进度

### 自动计算公式
- "用例汇总"Sheet的"完成数量"列使用公式自动统计
- "完成率"列自动计算百分比
- 无需手动更新，实时反映走查进度

### 筛选和排序
1. 点击表头的下拉箭头
2. 可以按优先级、检查点、设计原则等筛选
3. 可以按用例编号、优先级等排序

### 查找功能
- 按 `Ctrl+F`（Windows）或 `Cmd+F`（Mac）
- 可以快速查找特定的检查点或关键词

## 团队协作

可以将不同Sheet分配给不同的走查人员：
- 人员A：负责"跨域训练首页"和"新建任务"


### Excel转CSV
如果需要将Excel转回CSV格式（用于版本控制）：

```python
import pandas as pd

# 读取Excel的所有Sheet
excel_file = 'UI用例/跨域训练-UI走查用例-1.xlsx'
xls = pd.ExcelFile(excel_file)

# 合并所有Sheet（除了用例汇总）
all_data = []
for sheet_name in xls.sheet_names:
    if sheet_name != '用例汇总':
        df = pd.read_excel(xls, sheet_name)
        all_data.append(df)

# 合并并保存为CSV
combined_df = pd.concat(all_data, ignore_index=True)
combined_df.to_csv('UI用例/跨域训练-UI走查用例-合并.csv', index=False, encoding='utf-8')
```

## 常见问题

### Q1: Sheet名称显示不完整？
A: Excel的Sheet名称限制为31个字符，过长的名称会被自动截断。

### Q2: 如何添加新的用例？
A: 在对应的Sheet中，在最后一行下方插入新行，填写用例信息。注意"是否通过"列需要手动添加下拉选择（数据验证）。

### Q2.1: 为什么"是否通过"列没有下拉选择？
A: 如果新增行没有下拉选择，可以：
1. 选中有下拉选择的单元格
2. 复制（Ctrl+C）
3. 选中新增的单元格
4. 粘贴（Ctrl+V）

### Q3: 如何删除某个模块的Sheet？
A: 右键点击Sheet标签，选择"删除"，同时记得更新"用例汇总"Sheet。

### Q4: 如何导出为PDF？
A: 文件 → 另存为 → 选择PDF格式 → 可以选择导出所有Sheet或当前Sheet。

### Q5: 如何打印？
A: 建议按Sheet分别打印，每个Sheet设置为横向打印，调整缩放比例以适应纸张。
