# 用户登录功能需求文档

## 1. 功能概述
实现用户登录功能，支持账号密码登录和手机号验证码登录。

## 2. 页面结构

### 2.1 登录页面
- Logo和标题
- 登录方式切换（账号登录/手机登录）
- 输入框（账号/手机号、密码/验证码）
- 记住我选项
- 登录按钮
- 忘记密码链接
- 注册链接

### 2.2 忘记密码页面
- 手机号输入
- 验证码输入
- 新密码设置
- 确认按钮

## 3. 字段说明

| 字段 | 类型 | 说明 | 验证规则 |
|------|------|------|----------|
| 账号 | 文本 | 用户名或邮箱 | 4-20字符 |
| 密码 | 密码 | 登录密码 | 6-20字符，包含字母和数字 |
| 手机号 | 文本 | 11位手机号 | 手机号格式验证 |
| 验证码 | 文本 | 6位数字 | 6位数字 |

## 4. 交互说明

### 4.1 账号登录流程
1. 用户输入账号和密码
2. 点击登录按钮
3. 系统验证账号密码
4. 验证成功跳转首页，失败显示错误提示

### 4.2 手机登录流程
1. 用户输入手机号
2. 点击获取验证码
3. 输入验证码
4. 点击登录按钮
5. 验证成功跳转首页

## 5. 异常处理

- 账号不存在：提示"账号不存在"
- 密码错误：提示"密码错误，还可尝试X次"
- 验证码错误：提示"验证码错误"
- 验证码过期：提示"验证码已过期，请重新获取"
- 网络异常：提示"网络连接失败，请重试"
